# Box

A simple tutorial for implementing a React-like library. Uses `snabbdom` behind the hood. 

## How to use this?

1. Clone the repository

```sh
git clone https://github.com/rishiosaur/box
```

2. Install the dependenices

```sh
yarn
```

3. Run the application

```sh
yarn start:dev
```
